Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Kontynuujmy z rozwiązaniem z podpowiedzi poprzedniej misji w serii. 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    const cond2 = /\d/.test(password);
    
    return!([cond1, cond2].includes(false));
}

W tej misji należy dodać do rozwiązania sprawdzenie, czy hasło zawiera nie tylko cyfry.

Od jakiego instrumentu powinienem zacząć?

Zawierający nie tylko cyfry w tym przypadku oznacza zawierający co najmniej jedną literę. Może Pan powtórzyć wyrażenie dla poprzedniego warunku, używając non-digit metacharacter. Zwróci ono true, jeśli znak nie jest cyfrą. 

const cond3 = /\D/.test(password);

Teraz muszę spełnić wszystkie warunki true, aby rozwiązać zadanie. Jak to zrobić?

Proszę dodać cond3 wewnątrz tablicy w return: 

return!([cond1, cond2, cond3].includes(false));

Jak powinno wyglądać moje ostateczne rozwiązanie dla tej misji?

Oto ono: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    const cond2 = /\d/.test(password);
    const cond3 = /\D/.test(password);
    
    return!([cond1, cond2, cond3].includes(false));
}
