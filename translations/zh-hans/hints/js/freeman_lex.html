在这种情况下，不仅包含数字，还意味着至少包含一个字母。你可以重复前一个条件的表达，使用non-digit metacharacter 。如果字符不是数字，它将返回true 。 

const cond3 = /\D/.test(password);

现在我需要所有的条件都是true 来解决这个任务。如何做到这一点？

在return ，在数组中加入cond3 ： 

return!([cond1, cond2, cond3].includes(false));

那么我这个任务的最终解决方案应该是怎样的呢？

这就是了： 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    const cond2 = /\d/.test(password);
    const cond3 = /\D/.test(password);
    
    return!([cond1, cond2, cond3].includes(false));
}
