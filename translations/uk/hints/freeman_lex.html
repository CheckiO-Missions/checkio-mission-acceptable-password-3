Давай продовжимо розв'язок у підказках з попередньої місії в серії. 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(map(str.isdigit, password))

    return all([cond1, cond2])
У цій місії тобі слід додати до розв'язку перевірку чи пароль містить не тільки цифри.
З якого інструменту мені варто почати?
Мість не тільки цифри означає, що містить хоча б одну букву. Ти можеш повторити вираз з попередньої умови, з використанням рядкового методу str.isalpha(). Він повертає True, якщо символ є буквою. 

cond3 = any(map(str.isalpha, password))
Тепер мені потрібно, щоб всі умови були True, щоб розв'язати цю місію Як це зробити?
Додай cond3 всередину списку, який є аргументом функції all() в return: 

return all([cond1, cond2, cond3])
То як мій остаточний розв'язок має виглядати?
Ось він: 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(char.isdigit() for char in password)
    cond3 = any(char.isalpha() for char in password) 

    return all([cond1, cond2, cond3])

Використай lambda вираз для компактного вигляду, і декілька порівнянь. Оскільки c.isdigit() повертає логічне значення, яке може бути інтерпритоване як цілі числа 0 чи 1, функція sum() рахує кількість цифр у рядку. Вона повинна бути більшою за 0 (цифра присутні в рядку) і меншою за довжину рядка (не всі символи в рядку є цифрами). 

is_acceptable_password = lambda p: 0 < sum(c.isdigit() for c in p) < len(p) > 6
Якщо тобі можуть бути потрібні всі варіанти перевірки, ти можеш використати такий каскадний стиль розв'язку: 

def is_acceptable_passwordI(x: str) -> bool:
    return len(x) > 6

def is_acceptable_passwordII(x: str) -> bool:
    return is_acceptable_passwordI(x) and any([c in '0123456789' for c in x])

def is_acceptable_password(x: str) -> bool:
    return is_acceptable_passwordII(x) and not all([c in '0123456789' for c in x])
